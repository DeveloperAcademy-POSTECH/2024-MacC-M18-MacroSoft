## Convention
**타입**
* feat: 기능, UI

* fix: 버그

* ref: 리팩토링

* style: 코드 띄어쓰기•주석•정렬

* chore: 패키지, 환경설정, 프로젝트 파일

* docs: 문서

* test: 테스트

</br>

**Issue**: [타입/#이슈번호] 설명
>[FEAT/#1] 기능 추가

**Branch:**: 타입/#이슈번호/작업명
>feat/#1/기능 추가

**Commit**: 타입: #이슈번호 작업내용 설명
>feat: #1 기능 추가

**PR**: [타입/#이슈번호] 설명
